# 2026-05-10 프로젝트명 선정 회의록 — FINZ

> 우정주 MVP 프로젝트의 이름 후보를 모으고, 2단계 투표를 통해 최종 프로젝트명을 결정한 기록입니다.

## 핵심 결과

우정주 MVP 프로젝트의 최종 이름을 **FINZ** 로 결정했다.

FINZ는 Finance, Friends, Z세대 감성을 섞은 짧은 이름으로, 금융을 너무 딱딱하게 다루지 않고 친구들과 트렌디하게 즐기는 커뮤니티 앱의 방향과 잘 맞는다는 평가를 받았다.

## 선정 방식

프로젝트명 후보가 많아질 수 있어, 다음 방식으로 후보를 압축하고 최종 이름을 정했다.

1. 한 사람당 후보 3개씩 제출한다.
2. 1차 투표: 총 9개 후보에 대해 한 사람당 3개씩 투표한다.
3. 1차 투표에서 2표 이상 받은 후보를 2차 투표 후보로 선정한다.
4. 2차 투표: 최종 후보에 대해 한 사람당 1개씩 투표한다.
5. 가장 많은 표를 받은 이름을 최종 프로젝트명으로 결정한다.

## 제출 후보

### 남덕우

#### FINZ / 핀즈

Finance + Friends + Z세대 감성을 섞어서 만든 이름.

- 금융을 어렵지 않고 트렌디하게 즐기는 느낌
- 친구들과 종목 공유하며 노는 커뮤니티 감성
- 유니콘 스타트업 같은 짧고 세련된 분위기

#### Stock Slayer / 스톡슬레이어

Stock + Slayer를 합쳐서 만든 이름.

- 주식을 게임처럼 공략하는 도전적인 느낌
- 랭킹, 캐릭터, 배지 시스템과 잘 어울림
- 강렬하고 한번 들으면 기억에 남는 스타일

#### Loopy / 루피

Loop + playful 느낌을 합쳐서 만든 이름.

- 친구들이랑 계속 연결되는 투자 커뮤니티 느낌
- 딱딱한 주식 앱보다 재밌고 가벼운 SNS 감성
- 짧고 귀여운 캐릭터 느낌

### 김태훈

#### PlayStock

주식을 즐긴다는 의미의 이름.

#### MoonMates

친구랑 같이 달에 간다는 느낌의 이름. "To the Moon" 밈을 참고했다.

#### 10Baggers

미래의 텐배거 종목을 찾는 사람들이라는 의미의 이름.

### 김지헌

#### Rally Picks

친구들과 함께 픽하고 시장을 랠리한다는 의미의 이름.

#### STalk

Stock + Talk를 합친 언어유희.

#### Folio Chat

포트폴리오와 채팅을 결합한 이름.

## 1차 투표

총 9개 후보에 대해 각자 3개씩 투표했다.

2표 이상을 받은 후보:

- **FINZ**
- **Rally Picks**

위 두 후보를 2차 투표 후보로 선정했다.

## 2차 투표

1차 투표에서 선정된 두 후보에 대해 각자 1개씩 투표했다.

최종 결과:

- **FINZ**: 만장일치
- Rally Picks: 0표

## 최종 결정

최종 프로젝트명은 **FINZ** 로 결정했다.

| 항목 | 값 |
|---|---|
| 영문명 | FINZ |
| 한글명 | 핀즈 |
| 디렉터리명 후보 | `finz` |
| 의미 | Finance + Friends + Z세대 감성 |
| 결정 방식 | 2차 투표 만장일치 |

## 결정 이유

- 짧고 기억하기 쉽다.
- 직접적으로 "주식"이나 "투자"를 말하지 않지만 금융 맥락을 은근히 담고 있다.
- 친구 기반 투자 커뮤니티라는 방향과 잘 맞는다.
- MVP 이후 기능이 대화, 캐릭터, 모의투자, 리포트 등으로 확장되어도 이름이 특정 기능에 갇히지 않는다.
- 영어 프로젝트 디렉터리명으로 쓰기에도 간결하다.

## 후속 작업

- 새 프로젝트 또는 기능 영역을 만들 때 `finz`를 디렉터리명 후보로 사용한다.
- 기존 우정주 MVP 문서에서 프로젝트명을 언급할 때 FINZ/핀즈 명칭을 반영할지 검토한다.
- 제품 문서에서는 초기에는 "FINZ(핀즈)"처럼 영문명과 한글명을 병기한다.
