# 2026-05-02 회의 성과 요약

## 핵심 성과

GitHub와 Vercel을 활용한 협업 개발 워크플로우를 실습하고, PR(Pull Request)부터 배포까지 전체 파이프라인을 확인했다. 새로운 프로젝트로 우정주(친구들과 함께하는 투자) 관리 앱을 기획하기로 결정했으며, 실사용 가능한 MVP 개발을 목표로 방향을 설정했다.

## 주요 결정사항

- **프로젝트 방향 전환**: 여행 계획 앱(Travel)에서 실제 사용 빈도가 높은 투자 관련 앱으로 전환
- **MVP 핵심 기능**: 사용자들의 투자 성향과 관심사를 입력받아 LLM이 적합한 우정주 종목을 추천하는 기능
- **앱 컨셉**: 투자의 재미와 정보 공유를 결합한 채팅 기반 SNS 형태
- **개발 환경**: GitHub 협업 워크플로우 사용, PR-Review-Approve-Merge 프로세스 적용

## 기술 구현 사항

### 협업 워크플로우 확립

- **PR 생성**: 로컬에서 코드 수정 후 GitHub에 Pull Request 생성
- **Review 단계**: 팀원들이 변경사항 검토 (선택사항)
- **Approve**: 코드 승인 후 GitHub에 병합 가능
- **Merge**: 실제 코드 병합 시 Vercel에서 자동 배포 트리거
- **Preview 링크**: PR 생성 시 Vercel이 자동으로 미리보기 링크 제공
- **Production 배포**: Merge 완료 후 운영 환경 자동 배포

### 기술 이슈 해결

- **GitHub CLI 미설치 문제**: Codex 환경에서 gh 명령어 실행 실패, MCP를 통한 우회 시도
- **인증 문제**: HTTPS 인증 및 SSH 키 설정 필요
- **Repository 이전**: 개인 repo에서 organization repo로 이전하여 협업 환경 구축

## 우정주 앱 기획 내용

### 핵심 기능 (MVP)

- **채팅방 생성**: 투자 주제별 대화방 생성 (예: "우정주 찾기")
- **성향 입력**: 각 사용자의 투자 성향과 관심 섹터 입력
- **종목 추천**: LLM이 입력된 정보를 바탕으로 적합한 종목 추천
- **대화 기반 분석**: 채팅 내역을 분석하여 추가 가이던스 제공

### 향후 확장 기능

- **실시간 수익률 추적**: 투자 후 수익률 자동 계산 및 알림
- **투표 기능**: 중요한 시점에 투자 결정 투표
- **보상 시스템**: 수익 달성 시 채팅방 내 보상 기능
- **연간 리포트**: 1년간의 투자 기록 및 성과 요약
- **다중 그룹**: 관심사별로 여러 우정주 그룹 생성 가능

### 기술적 고려사항

- **LLM 비용 최적화**: 메시지마다 호출하지 않고 수동 트리거 또는 선택적 호출
- **데이터 보안**: 민감한 자산 정보는 로컬 저장, DB 미사용 검토
- **API 연동**: 주식 정보 API 연동 필요 여부 추후 결정
