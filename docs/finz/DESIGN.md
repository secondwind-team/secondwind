# FINZ 디자인 시스템 — 따뜻한 채팅 (Warm Chat)

> FINZ 전용 디자인 시스템. **secondwind / travel 과 완전히 독립**이다 — 토큰 네임스페이스(`--fz-*`), 전용 폰트, 전용 레이아웃 셸을 쓰고 공용 `--ink/--muted/--line/--accent` 나 emerald 팔레트를 절대 참조하지 않는다. UI/디자인 결정 전 이 문서를 먼저 읽는다.
>
> 2026-06-14 `/design-consultation` 으로 생성(미감 방향 "따뜻한 채팅" 확정).
> **2026-06-15 갱신: finz 가 단계별 화면에서 진짜 카카오톡식 채팅방으로 재구조화됨 + `@finz` 멘션 AI 답변 추가.** 시각 토큰(폰트·컬러·모양·모션)은 그대로, **정보구조가 "단일 채팅 타임라인"으로 바뀜**. 아래 [정보구조](#정보구조-카카오톡식-채팅방)·[채팅 메시지 종류](#채팅-메시지-종류)·[결정 로그](#결정-로그)를 먼저 본다.

## 제품 맥락

- **무엇:** 친구 2명이 **하나의 채팅방**에서 투자 취향 캐릭터로 만나, '오늘의 우정주'(테마)·각자의 한 줄 입장·AI 요약을 주고받고, 그 사이 **자유롭게 대화**하며, `@finz` 로 부르면 AI 가 오늘 주가·뉴스까지 답하는 **메신저형 투자 대화 앱**.
- **핵심 모델:** **단일 채팅 타임라인이 곧 상태**다. "저장" 버튼 없음 — 캐릭터 소환·우정주·입장·요약·자유 대화·`@finz` 답변이 전부 한 타임라인에 append 로 쌓인다.
- **누구:** Z세대 친구 그룹 (비개발자 포함). 증권 지식 없어도 부담 없이.
- **톤 원칙:** "사세요가 아니라 얘기해보세요." 투자 조언 아님, 대화 소재. (`@finz` 답변도 끝에 "정보 참고용" 면책을 서버가 강제 부착.)
- **타입:** 모바일 우선 **풀하이트 메신저**. (대시보드/유틸/단계별 폼 아님.)

## 기억에 남길 한마디

> **"친구랑 편하게 수다 떠는 곳."**

모든 디자인 결정은 이 한 문장에 복무한다. 차갑고 정밀한 핀테크가 아니라, 따뜻하고 친근한 메신저. 부담을 낮추는 게 1순위.

## 미감 방향

- **방향:** 따뜻한 채팅 (KakaoTalk × 토스 라이트).
- **장식 수준:** intentional — 큰 라운드, 부드러운 그림자, 절제된 이모지. 패턴/그라데이션 남용 없음.
- **무드:** 크림빛 종이 위의 말풍선. 둥글고 폭신하고 친근. 톡톡 튀지만 유치하지 않게.
- **레퍼런스:** KakaoTalk(말풍선·채팅 IA), Toss(둥근 카드·친근한 카피·바운시 모션).

## 타이포그래피

- **디스플레이:** **Cabinet Grotesk** (800/700) — 둥글고 친근한 지오메트릭. 헤더·캐릭터 클래스명·우정주 제목.
- **본문/UI:** **Pretendard** — 한글 가독성 최상. 본문, 라벨, 칩, 입력.
- **숫자(스탯 등):** Pretendard tabular-nums (별도 모노 안 씀 — 따뜻한 톤 유지).
- **로딩:** Cabinet Grotesk = Fontshare CDN, Pretendard = jsDelivr CDN. finz 레이아웃에서만 로드.
- **스케일(rem):** 12 / 14 / 15(본문) / 18 / 22 / 28 / 36(디스플레이). 행간 본문 1.5, 디스플레이 1.15, letter-spacing 디스플레이 -0.02em.

## 컬러

`--fz-*` 네임스페이스. emerald 계열 금지(공용과 시각 충돌).

- **배경:** `--fz-bg #FBF7F0` (따뜻한 크림)
- **서피스:** `--fz-surface #FFFFFF`, `--fz-surface-2 #FFF3EC` (코랄 틴트)
- **잉크:** `--fz-ink #2B2622` (소프트 다크브라운, 순검정 아님), `--fz-muted #8A7F76`
- **라인:** `--fz-line #EFE7DB`
- **액센트(코랄):** `--fz-coral #FF6B5C`, 텍스트용 `--fz-coral-ink #C8412F`, 틴트 `#FFEDE9`
- **서브(앰버):** `--fz-amber #FFC24B`, 틴트 `--fz-amber-tint #FFF6E4`, 텍스트 `#B6791B`
- **시맨틱:** success는 앰버계, error `#E05B4B`(코랄 다크), info는 코랄. 별도 채도 높은 색 추가 금지.
- **다크모드:** V0 미지원(라이트 단일). 추후 surface 재설계 시 채도 -15%.

## 모양 / 모션

- **라운드:** `--fz-r-sm 14` / `--fz-r 20` / `--fz-r-lg 28`(말풍선·큰 카드) / `--fz-r-full 999`(칩·버튼·아바타).
- **그림자:** 부드럽고 따뜻하게. `--fz-shadow 0 8px 24px -12px rgba(120,80,40,.22)`, `--fz-shadow-sm 0 3px 10px -6px rgba(120,80,40,.25)`.
- **모션:** 바운시·친근. 버튼/칩 탭 시 살짝 눌림(scale .96), 말풍선 등장 시 아래에서 톡 떠오름(translateY+fade, ease-out 220ms). 과한 choreography 없음.
- **easing:** enter `cubic-bezier(.2,.8,.2,1)`, exit ease-in. duration micro 90ms / short 180ms / medium 260ms.

## 정보구조 (카카오톡식 채팅방)

**단일 채팅 타임라인 = 화면 = 상태.** 더 이상 "단계별 화면"이 아니다. 방을 열면 하나의 메신저 화면이고, 모든 일이 메시지로 위에서 아래로 쌓인다.

**풀하이트 메신저 셸(멤버 전용):**

- **상단 헤더(고정):** 멤버 아바타(겹친 원형) + 상태 라벨(`친구를 기다리는 중` / 픽 후엔 테마명) + 초대 복사 버튼. (옛 "멤버 카드 그리드 + 초대 카드"를 한 줄로 압축.)
- **가운데 타임라인(스크롤):** 메시지가 쌓이는 본문. 내 행동엔 항상 바닥으로 따라가고, 상대/봇 메시지는 바닥 근처일 때만(아니면 "새 메시지" 칩). `aria-live` 로 스크린리더 간결 알림.
- **하단 입력바(고정):** 텍스트 입력 + 전송 + `+` 액션 시트(우정주 뽑기 / 내 입장 / 요약). 모바일 키보드는 `visualViewport` 로 추적해 입력바가 키보드 뒤로 안 숨음. placeholder 에 `@finz 로 질문` 힌트.

**비멤버 화면(풀블리드 아님):** 초대받은 비멤버는 전체 화면 "조인 뷰"(캐릭터 빌더, 자체 스크롤). 이미 2명이 찬 방이면 "새 파티 만들기" 안내만(타임라인 비노출).

**"이제 뭐 하지" 코칭(ephemeral nudge):** 현재 상태에서 다음 행동(초대 → 우정주 뽑기 → 내 입장 → 요약)을 클라이언트가 매 렌더 계산해 **타임라인 맨 아래 단 하나의 finz 코칭 말풍선 + CTA 버튼**으로 보여준다. **저장 안 함**(메시지 아님) — 상태가 진행되면 자연히 사라진다. 사용자의 "뭘 해야 할지 모르겠다"를 푸는 장치.

## 채팅 메시지 종류

타임라인에 쌓이는 메시지(append-only). 정렬·스타일이 발신자에 따라 다르다:

| kind | 발신 | 정렬 / 스타일 |
|---|---|---|
| `text` (멤버) | 나 / 상대 | 나=오른쪽 코랄 `.fz-msg--me`, 상대=왼쪽 흰 `.fz-msg` + 이름 라벨 |
| `text` (finz) | finz 봇 | 왼쪽 + 🤖 아바타 + "FINZ" 라벨 — `@finz` 질문 답변(그라운딩, 출처·면책 포함) |
| `system` | 시스템 | 가운데 작은 회색 pill ("OO님이 들어왔어요" 등) |
| `pick` | finz 봇 | 왼쪽 + 🤖 아바타 + `.fz-bubble--pick`(코랄 틴트) 큰 우정주 카드 + 재추첨 ghost 버튼 |
| `position` | 멤버 | text 처럼 좌/우 정렬, stance 이모지 칩 + 코멘트. 같은 사람의 옛 입장은 흐리게, 최신은 "바뀐 입장" 태그 |
| `summary` | finz 봇 | 왼쪽 + 🤖 아바타 + `.fz-bubble--amber`(앰버 틴트) 마무리 말풍선 |

- **캐릭터 = 프로필/스티커:** 헤더의 겹친 원형 아바타(클래스 이모지) + 조인 뷰의 풀 캐릭터 카드(스탯 미니바).
- **`@finz` = 호출형 봇:** 메시지에 `@finz`(또는 `@핀즈`)를 넣으면 finz 가 답한다. 답이 오는 동안 왼쪽에 `.fz-typing`(점 3개) 표시.
- **stance 칩:** 알약형, 선택 시 코랄 채움 + 그림자. (입력바의 "내 입장" 모드에서 노출.)

## 컴포넌트 토큰(핵심)

`app/finz/finz-theme.css` 의 재사용 클래스(전부 `.finz-root` 스코프, `--fz-*` 만 사용):

- **버튼 `.fz-btn` / `.fz-btn--ghost`:** primary = 코랄 채움 + 코랄 그림자, ghost = 흰 배경 + 라인. 모두 `--fz-r-full`.
- **칩 `.fz-chip`(stance/태그):** 알약형, 1.5px 라인, 선택 시 코랄.
- **카드/말풍선 `.fz-card` / `.fz-bubble`(`--pick` 코랄·`--amber` 앰버):** 흰 서피스 + `--fz-line` 1px + `--fz-shadow-sm`, 라운드 `--fz-r-lg`. `.fz-bubble` 은 등장 모션(fzPop).
- **채팅 말풍선 `.fz-msg` / `.fz-msg--me`:** 좌측 흰색 / 우측 코랄(흰 글자), max-width 80%, 등장 모션. **채팅 타임라인의 기본 단위.**
- **타이핑 인디케이터 `.fz-typing`:** 점 3개 깜빡임(fzBlink). finz 가 픽/요약/`@finz` 답변을 만드는 중 표시. reduced-motion 시 정지.
- **스탯바 `.fz-statbar` / 아바타 `.fz-avatar` / 태그 `.fz-tag` / 입력 `.fz-input` / 알림 `.fz-alert`.**
- **모션 접근성:** `@media (prefers-reduced-motion: reduce)` 에서 말풍선/타이핑 애니메이션·부드러운 자동 스크롤 끔.

## 기술적 독립 (아키텍처)

travel/secondwind 와 시각·구조 모두 분리:

1. **레이아웃 셸 분리:** finz 페이지를 `app/(site)/finz/` → **`app/finz/`** 로 옮겨 자체 `app/finz/layout.tsx` 를 둔다. 공용 `app/(site)/layout.tsx` 의 상단 nav(travel/FINZ 링크)를 **상속하지 않음**. URL 은 `/finz` 그대로(route group 변경이라 경로 불변).
2. **토큰 네임스페이스:** finz 컴포넌트는 `--fz-*` 만 쓴다. 공용 `--ink/--muted/--line/--accent`·emerald-* 참조 금지. (네임스페이스만으로도 cascade 격리됨.)
3. **전용 폰트:** Cabinet Grotesk / Pretendard 는 finz 레이아웃에서만 로드. 공용 폰트와 안 섞임.
4. **전용 globals:** `app/finz/finz-theme.css`(또는 layout 내 style)에서 `--fz-*` 정의. 공용 globals 의 토큰을 덮어쓰지 않고 별도 네임스페이스로 공존.
5. **풀하이트 셸:** `.finz-root` 는 `flex flex-col` + `min-height:100svh`, `app/finz/layout.tsx` 의 `<main>` 은 `flex-1 min-h-0` 라 채팅방이 (뷰포트 − 헤더) 높이를 정확히 채운다(매직넘버 calc 없음). 홈·생성 페이지는 각자 패딩을 가짐.

### 채팅(상태) 아키텍처 — 디자인 함의

UI 작업 전 알아둘 데이터 모델(상세는 코드 주석 참조):

- **2-키 이벤트 로그:** `sw:finz:group:<id>`=신원(멤버)만, `sw:finz:chat:<id>`=메시지 Redis LIST(원자적 append). 화면은 이 LIST 를 폴링(보임 3s/숨김 8s)해 그린다. → **모든 UI 는 "append-only 타임라인"을 전제로 설계**(되돌리기·편집 없음, 새 상태는 새 메시지).
- **`@finz` 답변:** `POST .../ask` 가 Google Search 그라운딩으로 실시간 답변 생성(픽은 theme-only 환각 방어 유지, 그라운딩은 `@finz` Q&A 에만). 면책 문구는 서버 불변식.

## 적용 대상 (현재 화면/컴포넌트)

- **화면:** `/finz`(시작·취향카드·캐릭터 소환), `/finz/party`(생성), `/finz/party/[groupId]`(**채팅방** — 핵심).
- **채팅방 컴포넌트:** `finz-party-room`(오케스트레이터·폴링·낙관적 전송·nudge·`@finz` 트리거), `finz-chat-header`, `finz-chat-timeline`, `finz-chat-message-view`(kind별 렌더), `finz-chat-composer`(입력바·액션시트·입장 모드), `finz-nudge-bubble`, `finz-position-input`, `finz-join-view`, `finz-room-full-notice`.
- **재사용:** `finz-character-card`(헤더 아바타·조인 뷰), `finz-party-pick-result`(픽 말풍선 내부), `finz-party-summary`(요약 말풍선 내부), `finz-character-builder`·`finz-taste-selector`(개인 흐름·조인).
- **폐지:** `finz-party-positions`(타임라인 position 메시지 + composer 입장 모드로 대체).
