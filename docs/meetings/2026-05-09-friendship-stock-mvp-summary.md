# 2026-05-09 우정주 MVP 미팅 Zoom AI 요약

## 핵심 결과

친구들과 투자 취향을 공유하며 즐겁게 소통하는 게임형 투자 커뮤니티 앱의 MVP 1.0 방향을 확정했다. 수익률이 아닌 투자 캐릭터와 대화 중심의 경험을 제공하며, 넷플릭스식 취향 선택으로 시작해 AI가 추천하는 '오늘의 우정주'를 중심으로 친구들이 토론하는 구조다.

## 주요 결정사항

### 제품 핵심 컨셉

- 목적: 돈을 버는 것이 아닌 친구들과 투자 맥락에서 즐겁게 소통하는 것
- 첫 경험: 종목 검색/차트가 아닌 넷플릭스식 취향 카드 선택으로 진입 장벽 낮춤
- 차별점: 증권앱이 아닌 투자 대화를 RPG처럼 만드는 게임형 커뮤니티

### MVP 1.0 기능 범위

- 캐릭터 생성: 추상적 취향 카드 선택 → 게임 캐릭터 형태로 투자 성향 시각화
- 캐릭터 구성: 클래스명, 한 줄 설명, 6가지 스탯, 약점, 공유하고 싶은 한 줄 놀림 포함
- 그룹 기능: 친구 초대 시 각자의 캐릭터가 파티처럼 표시
- 핵심 콘텐츠: AI가 추천하는 '오늘의 우정주' + 투자 레이드(토론)
- 텍스트 기반: MVP 1.0은 수익률 추적/실제 매매 없이 텍스트 기반 예측과 포지션 선택으로 구현

### 향후 확장 기능 (MVP 1.1~)

- 1.1: 예측 카드 기능 - 한 주 예측 후 결과 반영, 캐릭터 성장 로그
- 1.2: 파티 모의투자 - 공동 가상 포트폴리오, 투표로 편입/비중 결정
- 추가 고려사항: 투표 기능, 개인 모의투자 리그, 주간 파티 리포트, 캐릭터 성장 시스템

### 기술적 접근

- 개발 환경: VS Code + Cursor 플러그인 사용, 각자 토이 프로젝트로 익숙해지기
- AI 활용: 코덱스에게 비개발자 입장에서 질문하며 사전 세팅 진행
- 토큰 관리: 스킬/에이전트 사용 시 토큰 소모량 급증 주의 필요

## 주요 논의 내용

### 캐릭터 시스템

- 캐릭터 타입: 게임 RPG처럼 딜러/탱커/힐러 등 역할 부여
- 시각화: 게임 캐릭터 에디터처럼 등장, 도트 픽셀 방식도 고려
- 성장 메커니즘: 채팅 기록 기반으로 AI가 성향 분석해 캐릭터 업데이트
- 히스토리: 과거 투자 성향 변화를 캐릭터 진화로 기록

### 우정주 운영 방식

- 정의: 그룹의 공통 관심사를 반영한 종목, 함께 토론할 주제
- 목적: 공동 투자가 아닌 대화 소재 제공
- AI 역할: 그룹 성향 분석 후 매일/주간 우정주 추천
- 대화 유도: 우정주에 대한 각자 의견(찬성/반대/관망) 제시하며 토론

### 모의투자 접근

- 합의 방식: 그룹 합의로 가상 포트폴리오 운영, 개인은 선택적 실제 투자 가능
- 매매 주기: 장기투자보다 단기(일주일 단위) 또는 데이트레이딩으로 피드백 빠르게
- 다양한 기법: AI가 매주 다른 투자 기법 제시해 실험
- API 연동: 외부 의존성 높지만 통합 API 찾아볼 예정

## 보류/추후 검토 사항

- 캐릭터 이미지 퀄리티 수준 (전신 vs 얼굴만, 렌더링 vs 도트픽셀)
- 실시간 수익률 추적 vs 주간 요약 방식
- 개인별 경쟁 vs 그룹 협력 중심 설계
- 채팅 내용 AI 자동 분석 vs 사용자 수동 태그 방식
