## 핵심 결과

AI 기반 메신저 플랫폼 '핀즈'의 주요 기능(우정주 뽑기, AI 트리거, 주가 차트 임베딩)을 실시간으로 시연하고, 경제 시황 브리핑 스케줄러 및 종목 집중 토론방 등 신규 기능 아이디어를 도출했다. 트레이딩뷰 차트 임베딩이 즉석에서 구현되었으며, 다음 단계로 GitHub Actions 기반 스케줄러 구현을 시도했다.

## 결정 사항

- **플랫폼 구조 확정**: 메신저 형태로 확정 (친구 목록 + 대화 목록 + 피드 탭)
- **AI 트리거 방식**: @핀즈 멘션 시 LLM 호출, 현재는 지침 없는 날것의 상태로 추후 프롬프트 가이드 정비 필요
- **트레이딩뷰 차트 임베딩**: API 키 불필요한 공개형 API 활용, 즉석 구현 완료
- **스케줄러 구현 방향**: Vercel 무료 플랜 하루 1회 제한으로 GitHub Actions 방식 채택
- **인프라 원칙**: 인프라 추가 없이 기존 구조에 기능을 붙이는 방식 유지

## 완료된 작업

- 주가 차트(테슬라, 비트코인) 채팅방 내 실시간 임베딩 구현
- 우정주(종목/테마 추천) 기능 동작 확인 — 종목 또는 테마 형태로 출력
- PWA 형태로 모바일 홈 화면 설치 방법 안내 완료

## 주요 아이디어 (미구현)

- **경제 시황 브리핑**: 매일 오전 9시, 당일 경제 시황 300자 요약 + 주요 종목 채팅방 자동 발송
- **구독형 컨텐츠**: 특정 종목/테마 브리핑을 패키징해 채팅방 단위로 구독 — 유료화 가능성 언급
- **집중 토론방**: 우정주로 종목 선정 후 해당 종목에 특화된 제한적 기능만 제공하는 별도 방 형태
- **차트 고정 표시**: 채팅 스크롤과 무관하게 화면 상단 고정 또는 공지 형태로 표시

## 미결 사항

- **LLM 프롬프트 지침(가이드) 미정비**: AI가 일부 요청 거절하는 현상 발생
- **안드로이드 PWA 알림 기능 미구현**: 알림 기능 구현 후 별도 안내 예정
- **방 구조 설계**: 집중 토론방 vs. 자유 대화방 구조 설계 방향 미확정
- **계정 관리 방식**: 인프라(API 키 등) 추가 시 계정 관리 방식 논의 필요
